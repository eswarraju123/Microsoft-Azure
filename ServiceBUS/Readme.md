# Microsoft Azure Service BUS

The Microsoft Azure Service BUS SDK allows you to build Azure applications that take advantage of scalable cloud computing resources.

## Features

- Queue
    - Create/Delete Queue
    - Queue/Dequeue Messages
- Topic
    - Create/Delete Topic
    - Queue/Dequeue Messages
- Relay
    - Create/Delete Relay
